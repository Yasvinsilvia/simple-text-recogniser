# Simple Text Recogniser - Sentiment Analysis

## What it does
This project detects if text expresses HAPPY 😊 or SAD 😢 emotions.

## Examples
| Input | Output |
|-------|--------|
| "This product is amazing" | HAPPY 😊 |
| "This is not good" | SAD 😢 |

## How to use
1. Open the Google Colab notebook
2. Run all cells
3. Enter your text when prompted
